# Practice project

---

[Base on kanban manager](https://github.com/gionny96/dnd-kanban-board), add wysiwyg editor and markdown support in Task tag

## Deps

---

- React v18
  - Typescript
- ChakraUI
- react-dnd
- usehooks-ts
- Tiptap

## Installation

```zsh
  pnpm install
  pnpm dev
```
